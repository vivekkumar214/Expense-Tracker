# 💰 Expense Tracker with Budget Insights

## 📌 Project Overview

Expense Tracker with Budget Insights is a browser-based web application developed using HTML, CSS, and JavaScript. It helps users manage their daily income and expenses, set a monthly budget, visualize spending through charts, and store data using Local Storage.

---

## 🚀 Features

- Add Income & Expense Transactions
- Edit Transactions
- Delete Transactions
- Monthly Budget Management
- Budget Warning Notification
- Search Transactions
- Filter by Transaction Type
- Filter by Date
- Export Transactions to CSV
- Dark Mode
- Responsive Design
- Pie Chart (Expense by Category)
- Bar Chart (Income vs Expense)
- Indian Currency Formatting
- Local Storage Support

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage API

## ▶️ How to Run

1. Download or Clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
4. Start adding your transactions.
